WORK

POKEDEX VIEWER

An interactive Pokédex search tool built on the PokéAPIv2 RESTful service, using VanillaJS and jQuery. Allows user to search any Pokédex version by Pokémon name or order number, and displays dynamically-generated information page of Pokémon stats

WIKIPEDIA SEARCH

Connects via AJAX call to MediaWiki API; loads preliminary results as user types; iterates through JSON data to construct clickable search results (links); also allows user to view "Random Search" page if desired

LOCAL WEATHER APP

Requests user location data from browser à la navigator.geolocation; makes AJAX calls to two API weather services, one for current conditions and one for forecast; displays weather data with corresponding icon and Fahrenheit/Celsius toggle
